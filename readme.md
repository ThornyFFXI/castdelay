# CastDelay
Castdelay stops a cast, ranged, or item usage attempt if you stop abruptly and the server believes you are still moving. It will reinject the action if you fully stop within 5 seconds of trying. Castdelay will not reinject the action if you perform another action in between or change zones.

# (Lu)Ashitacast
If you're using any of these equipment swapping systems, ensure that castdelay is later in the load order for best results. For LuAshitacast, this just means loading this addon after LuAshitacast. For AC/LegacyAC, you must load the addons plugin itself after them.
