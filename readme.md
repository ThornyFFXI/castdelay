# CastDelay
Castdelay stops a cast, ranged, or item usage attempt if you stop abruptly and the server believes you are still moving. It will reinject the action if you fully stop within 5 seconds of trying. Castdelay will not reinject the action if you perform another action in between or change zones. It is fully compatible with LuAshitacast. If you are using functions that alter a spell's Id in LAC, it is recommended you place CastDelay after LAC in the load order to ensure the change takes place.
